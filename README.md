# PETase_structure_activity
Goal: Train supervised models that accept PDB files as inputs and output enzymatic activity predictions for a large number of plastic degrading enzymes / PETases
Three models will be trained for three datasets with the potential for combining them later on: Seo (158 rows), Norton-Baker (214 rows), and Rodney's (not yet ready)
The overarching goal of this project / the scientific question at hand is: Given a large dataset of plastic degrading enzymes, can we predict find relationships between protein structure and ecological environment and between protein structure and enzymatic activity?
